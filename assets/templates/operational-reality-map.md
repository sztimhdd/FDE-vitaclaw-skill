# Operational Reality Map

## Use

Use when the team must reconstruct how work actually happens today.

## Required Fields

- Core actors
- Responsibility chain
- Current-state workflow
- Systems and channels in use
- Evidence artifacts
- Clean case replay
- Ugly case replay
- Exception paths
- Known failure points

## Optional Fields

- Shadow tools
- Timing and SLA notes
- Escalation path

## Typical Anti-Pattern

The map reflects the official SOP only and misses workaround behavior, exceptions, or manual handoffs.

## Completion Standard

A delivery lead should be able to replay one clean case and one ugly case end-to-end.
