# Customer Brief

We are a seafood export business. For each shipment, our team receives purchase orders, airway bills, health certificates, printed product labels, and a spreadsheet that describes which label fields need to be checked.

Today, staff manually compare the label against the source documents before shipment. The work is slow, repetitive, and easy to get wrong, especially when the same field appears in different documents with slightly different formats.

We want an AI-assisted tool that helps validate the printed labels against the correct source documents and shows discrepancies clearly.

## Initial Customer Wording

> We have PO, AWB, HC, labels, and a field checklist. We need AI to help check whether the label is correct before shipment.

## Important Boundary

The desired system validates labels. It does not generate final labels in the first version.
