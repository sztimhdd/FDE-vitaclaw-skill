# FDE (Applied AI Operator OS Alias)

`fde` is a shorthand alias for `$fde-operator-os`.

Use it when you want the shorter invocation name or when a runtime or user naturally reaches for `FDE` first.

## Usage

```text
$fde "task description"
```

This alias maps to the canonical skill:

```text

```

## Scope

This alias does not change the workflow, stages, or artifacts.

It only provides a shorter entry point for hosts that support skill wrappers or alternate invocation names.
